# Insertion Sort

Insertion sort går flytter det j-te elementet trinnvis ned i arrayet A, fram til A\[1, j - 1] er sortert korrekt.
Dette gjøres for j \[2, len(A)].

## Algoritme

Kjøretid:
- $O(n^2)$ worst case
- $O(n^2)$ average case
- Lineær ved små n
